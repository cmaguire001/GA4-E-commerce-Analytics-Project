# GA4-E-commerce-Analytics-Project

## Project Overview
Analysis of Google Merchandise Store data using Google Analytics 4 to uncover customer behavior insights and revenue optimization opportunities.

## Business Objectives
- Understand customer purchase patterns
- Identify high-performing products and categories
- Analyze traffic sources and conversion rates
- Provide actionable recommendations for revenue growth

## Tools & Skills Demonstrated
- Google Analytics 4
- Data Analysis
- Data Visualization
- Statistical Analysis
- Business Intelligence

## Dataset
Google Merchandise Store (GA4 Demo Account)
